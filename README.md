### Description

```
    Мобильное тестирование приложения app_debug.apk 
    с помощью, Appium и фремворка Selenide.
    В Данной работе так же используется STUB сервер.
```

### Install

```
    git clone https://github.com/justiwannauseit/home_task_6 
```

### Run

```
    В проекте используется два модуля:
    -> Сервис загршка в пакете stub
    -> Тестовый фреймворк
    .................................................................
    Для корректной работы тестового фреймворка необходимо поднять
    в первую очередь Stub сервер! Только после этого запускать 
    appium и avd emulator. После успешного запуска, можно запускать тесты.
    
    (!) Примечание: STUB сервер поднимается на http://127.0.0.1:8080/
    Мокирование проиходит по следующими методами-заглушами:
    -> получение статуса сервиса /ping-pong
    -> получение списка всех пользователей /users/get/all
    -> получение информации о пользователе по id /users/get/{id}
```
